# Project A: "Pathfinder" (Intelligent Academic Advising)

## Context
Students are struggling to graduate on time because they take the wrong prerequisites or cannot find open slots for mandatory courses. The university wants an automated roadmap builder.

## Raw Elicitation Notes
1. Interview with Head of Advising: "I spend 6 hours a day answering emails asking 'Can I take Course B if I have not done Course A?' The system needs to stop them from even asking. Also, I want to be able to override the system manually if a student has special permission."
2. Focus Group (Freshmen): "The course catalog is ugly. I just want to drag and drop classes into a calendar to see if they fit. I want the app to warn me if I am choosing a class that starts at 8:00 AM because I will definitely oversleep."
3. IT Security Officer: "Student data is sensitive. Under no circumstances should a student be able to see another student schedule. Also, the system must log every time an advisor manually overrides a prerequisite for audit purposes."
4. Feedback from Registrar: "The system needs to link to the live database. If a class is full, the student should not be able to add it to their roadmap. But the Advising Head said they want students to be able to plan 4 years ahead, even if we do not know if the class is full yet. That sounds contradictory."
